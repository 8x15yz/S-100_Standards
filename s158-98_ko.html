S-158:98 상호운용성 검증 규칙

Data Product Interoperability Validation Checks

S-158:98은 S-158 프레임워크를 따르는 파생 규격으로, 여러 S-100 데이터 제품을 함께 사용할 때의 데이터 간 호환성을 점검하기 위한 상호운용성 검증 규칙을 정의합니다.

표준 개요

S-158 프레임워크를 따르는 파생 규격으로, S-100 데이터 제품 간 상호운용성 검증 규칙을 정의하는 표준임. 정식 명칭은 Data Product Interoperability Validation Checks이며, Edition 1.0.0은 2025년 2월 발행됨.

개별 제품 내부의 무결성이 아닌, 여러 S-100 제품을 함께 사용할 때의 데이터 간 호환성을 점검하기 위한 규격임. S-158:100 및 S-158:1xx 검증을 보완하여 제품 간 상호운용성 검증 계층을 제공함.

적용목적 및 대상

목적

  • S-98에서 요구하는 상호운용성 조건을 검증 소프트웨어에서 구현할 수 있도록 검증 규칙을 제공함
  • ECDIS에서 함께 사용되는 데이터셋 및 exchange set의 교차 적합성(cross-compatibility) 검증을 지원함
  • S-158:100 및 S-158:1xx 검증을 보완하여 제품 간 상호운용성 검증 계층을 제공함

대상

  • 적용 제품: S-101, S-102, S-104, S-111, S-124, S-128, S-129
  • 개별 데이터셋 조합, exchange set, CATALOG.XML을 기반으로 하는 discovery metadata 포함
  • 주요 활용 주체: 검증도구 개발자, 데이터 생산·편집·집성·배포 시스템, ECDIS

버전 업데이트 타임라인

  • 2025.02 / Version 1.0.0 — 첫 구현 및 시험용 Edition 발행, S-100 WG 피드백 반영, 체크 번호 체계 업데이트, 오류 메시지에 오류 위치 식별 요구사항 추가
  • 2024.12.11 / Version 0.2.1 — DQ Measure 컬럼 제거, overlap 관련 체크 정제, S-102 holes check 추가, 분류 체계 C/E/W indicator 방식으로 정리
  • 2024.12.01 / Version 0.2.0 — maintenance section 업데이트, tolerance 업데이트, 일부 generic checks를 S-100 쪽으로 이동·정리
  • 2024.10.23 / Version 0.1.0 — 초기 초안 작성 (S-98 Annex C Edition 1.0.0 및 S-100 WG7 8.1 기반)

모델 구성

S-158의 기본 검증 구조를 따르되, 상호운용성 검증에 필요한 추가 필드를 확장하여 사용함. Data Quality Measure 컬럼은 S-158:98에서 사용하지 않음.

권장 검증 적용 순서

S-158:100 공통 검사 → S-158:1xx 제품별 검사 → S-158:98 상호운용성 검사(dataset) → S-158:100 exchange set 검사 → S-158:1xx exchange set 검사 → S-158:98 상호운용성 검사(CATALOG.XML) → S-158:128 카탈로그 검사

심볼, 테스트데이터셋 예시

S-158:98은 별도의 심볼 체계를 정의하지 않고, S-158의 Symbols를 그대로 참조함.

상호운용성 검증 항목 예시

  • S-101 / S-102 / S-104 간 수직 기준면·음향측심 기준 불일치 검사
  • S-102 / S-104 / S-111 격자형 coverage 데이터 중첩 검사
  • S-101 / S-124 / S-129 벡터 데이터셋 중첩 검사
  • CATALOG.XML 및 exchange set 리소스 정합성 검사
